# Digital System Design

A collection of RTL designs and digital circuits.

#  Table of Contents

| Module / File   | Description                                      | Key Contents |
|-----------------|--------------------------------------------------|--------------|
| Adders          | Implementation of different adder circuits       | Half Adder, Full Adder designs |
| FourBitAdder    | 4-bit ripple carry adder implementation          | Structural/behavioral HDL files |
| MUX             | Multiplexer circuit designs                      | 2:1, 4:1 MUX implementations |
| Decoders        | Binary decoder circuit implementations           | 2-to-4, 3-to-8 decoders |
| AsyncRCntr      | Asynchronous ripple counter design               | Counter modules & testbenches |
| PatternCounter  | Counter for detecting specific patterns          | Sequential logic design files |
| Sequential      | General sequential circuit examples              | Flip-flops, FSM examples |

# Part 3: Push-Button Up/Down Counter

- Required modules: `PushButtonDebouncer`, `SyncUpDownCounter`, `Part3_PBUpDownCounterTop`
- Top module: `Part3_PBUpDownCounterTop`
- Main inputs/outputs: `clk`, `reset_n`, `up_down`, `pb`, `count[3:0]`
- Description: debounces a push button and uses each debounced press to step a 0-9 up/down counter.
